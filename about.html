About Pocket Classroom

A simple, local-first classroom web app built with HTML, CSS and Vanilla JavaScript. Designed to be a starting point for learning full-stack features later.

Features

  • Dark modern UI
  • Lessons with video, description and assignments
  • Per-lesson quizzes and a quizzes page
  • Chat saved to LocalStorage
  • Export / Import JSON backup
  • Add / Edit / Delete lessons

Next steps

  1. Connect to a backend (Firebase) to enable multi-user and persistence
  2. Add file uploads for video and attachments
  3. Add user roles (teacher / student) and authentication
